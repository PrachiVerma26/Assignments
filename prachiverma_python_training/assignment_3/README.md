# Python Data Science and Visualization

### Overview

This repository contains solutions for Data Science and Visualization assignments implemented using Python and Jupyter Notebooks. The assignments focus on data analysis, data cleaning, and data visualization using NumPy, Pandas, Matplotlib, and Seaborn.

### Prerequisites

* Python 3.10+
* pip (Python package manager)

### Required Libraries

* **NumPy** - Numerical computing library for array operations
* **Pandas** - Data manipulation and analysis library
* **Matplotlib** - Data visualization library
* **Seaborn** - Statistical data visualization library built on Matplotlib
* **ipykernel** - Python kernel for running Jupyter notebooks

### Setup Instructions

#### Create a Virtual Environment

```bash
python -m venv venv
```

#### Activate the Virtual Environment

**Windows PowerShell**

```powershell
venv\Scripts\Activate.ps1
```

**Windows Command Prompt**

```cmd
venv\Scripts\activate.bat
```

#### Install Dependencies

```bash
pip install -r requirements.txt
```

#### Launch Jupyter Notebook

```bash
jupyter notebook
```
+
#### Launch Jupyter Lab

```bash
jupyter lab
```

Open and run the notebook files from the browser interface.

#### Using VS Code

Open the `.ipynb` files directly in VS Code with the Python and Jupyter extensions installed.

### Notes

* All assignments are implemented using Jupyter Notebooks.
* Execute notebook cells sequentially to ensure correct outputs.
* Install all required dependencies before running the notebooks.
* Visualizations are generated using Matplotlib and Seaborn.
