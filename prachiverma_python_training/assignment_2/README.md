# Python Advance Assignment 1

### Overview

This repository contains Python programs covering advanced Python concepts such as exception handling, testing and debugging, iterators, generators, functional programming, regular expressions, packaging, threading, and multiprocessing.

### Prerequisites

* Python 3.10+
* Visual Studio Code (Recommended)

### Project Structure

```text
assignment_2/
├── exception_handling/
├── testing_and_debugging/
├── iterators_and_generators/
├── functional_programming/
├── regular_expressions/
├── packaging/
├── threading_and_multiprocessing/
└── README.md
```

### Setup Instructions

#### Clone the Repository

```bash
git clone <repository-url>
cd assignment_2
```

#### Open in VS Code

```bash
code .
```

#### Run a Python Program

```bash
python filename.py
```

#### Example

```bash
python exception_handling/q7_custom_exception.py
```

### Learning Outcomes

* Understand advanced Python programming concepts.
* Implement exception handling and debugging techniques.
* Work with iterators and generators efficiently.
* Apply functional programming concepts.
* Use regular expressions for pattern matching and validation.
* Organize code using modules and packages.
* Execute tasks concurrently using threads and processes.
